# Roadmap

## v0.1.0

- Static repository risk scan before AI agents run code.
- CLI, JSON, Markdown, and terminal output.
- Web UI for public GitHub repository scans.
- Deterministic verifier and scoring.

## Next Priorities

1. Add more language-specific manifest scanners.
2. Add SARIF export.
3. Add richer MCP config detection.
4. Add optional signed rule bundles.
5. Add Docker sandbox experiments behind an explicit disabled-by-default flag.
