# Contributing

Thanks for helping improve VibeProof.

## Local Setup

```powershell
npm install
npm run verify
```

## Pull Request Checklist

- Add or update tests for scanner, verifier, CLI, or UI changes.
- Keep default scans static; do not execute target repository code.
- Mask secrets in reports and logs.
- Update `README.md` and `README.ko.md` together when user-facing commands change.
